# Installing and setting up an Editor

There are many amazing code editors to choose from that can improve your programming journey. However not all of them support Crystal out of the box ... or at all. Here's a list of them and the required Crystal plugins:

  • Atom (opens new window)
    • crystal-tools (opens new window) - Enables built in tools in Crystal compiler
    • language-crystal-actual (opens new window) - Crystal language support in Atom
  • Emacs (opens new window)
    • emacs-crystal-mode (opens new window) - Crystal language support for Emacs
  • scry (opens new window) - Code analysis server for Crystal implementing the Language Server Protocol (opens new window)
  • Komodo Edit (opens new window)
    • Krystal (opens new window) - A (very basic) Crystal-Lang Highlighter for Komodo Edit/IDE
  • Spacemacs (opens new window)
    • crystal-spacemacs-layer (opens new window) - Spacemacs contribution layer for Crystal
  • Sublime (opens new window)
    • sublime-crystal (opens new window) - Crystal syntax highlighting for sublime Text
  • TextMate (opens new window)
    • Crystal.tmbundle (opens new window) - Crystal syntax highlighting, compile, format command, snippets
  • Vim (opens new window)
    • vim-crystal (opens new window) - Vim filetype support for Crystal
    • vim-slang (opens new window) - Vim filetype support for Slang Templating Engine
  • Visual Studio Code (opens new window) | VSCodium (opens new window)
    • vscode-crystal (opens new window) - Crystal language support in VSCode
    • vscode-crystal-ide (opens new window) - Crystal IDE powered by Language Server Protocol (opens new window)
    • vscode-crystal-lang (opens new window) - Formatter, linter and syntax highlighting for cr and ecr files

from: https://github.com/veelenga/awesome-crystal (opens new window)

TIP

Use crystal tool format to format all your files!
